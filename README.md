# Java-MVC-DAO
mini-hospital-manager


Application Java simulant le système interne de gestion d’un petit hôpital.

 Objectifs
- Concevoir une application métier réaliste
- Mettre en œuvre l’architecture MVC
- Utiliser le pattern DAO pour la persistance

 Technologies
- Java 17
- Architecture MVC
- DAO (Data Access Object)
- JDBC + SQLite
- Exceptions personnalisées
- Streams & validations métier

 Fonctionnalités
- Gestion des patients et des médecins
- Planification des rendez-vous
- Détection des conflits de planning
- Séparation claire des couches (controller, service, DAO)

Ce projet démontre ma capacité à **structurer une application Java proche des
besoins réels d’une entreprise**.
